# Random-Task-Generator
Random Task Generator — это GUI‑приложение на Python для генерации случайных задач с возможностью сохранения истории и фильтрации по типам задач.
Основные функции:
Генерация случайной задачи из предопределённого списка.
Добавление новых задач пользователем.
Фильтрация задач по типам: учёба, спорт, работа.
